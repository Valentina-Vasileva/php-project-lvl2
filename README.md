GENDIFF
========

A PHP package which shows difference between two files (yaml, json).

Installation  
------------
Via Composer

    $ composer global require valentina-vasileva/php-project-lvl2

Usage
-----
   
    $ gendiff <path_to_file1> <path_to_file2>
